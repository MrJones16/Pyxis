# Pyxis
Pyxis Game Engine

## About
Pyxis is an open-source game engine written in C++ focused on advanced falling sand simulation features & multiplayer networking.

## Libraries/Packages/Frameworks
- SDL3
- GameNetworkingSockets
- Box2D
- Nlohmann Json
- Snappy

## More Details
This is V2 of the engine, and this version will be based on SDL3 and an ECS. 

The prior was written completely from scratch by me, and while I learned a lot, it was not the best implementation and it racked up too much tech debt. It was also very inheritance based which is not terrible but I want to go towards an actual engine, for the people!
